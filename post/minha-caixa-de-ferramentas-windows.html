Minha caixa de ferramentas

Published on Monday, June 8, 2020
toolbox

Quando falamos sobre uma "caixa de ferramentas" (e só para ficar bem claro, estamos falando de software), usuários de Linux normalmente tem seus setups muito bem definidos, o que acaba acontecendo meio que por osmose durante o uso de distros diferentes... Mas no fundo todos os usuários avançados de algum sistema operacional tem sua lista de ferramentas favoritas, que são utilizadas com propósitos diversos para otimizar sua experiência. Com Windows não é diferente, e esta é a minha lista.

Top 5 ferramentas

Essas são as principais ferramentas que eu uso no meu dia a dia. Não há um critério definido para as ferramentas listadas aqui além do meu próprio capricho. Em maioria, as ferramentas listadas devem ser utilizadas em Windows, mas atualmente meu uso está bem dividido entre um Manjaro (linux, arch-based) e Windows, de forma que a lista pode conter ferramentas utilizadas em mais de um sistema operacional.

  • Chocolatey: Uma opção para instalar softwares em Windows via commandline, semelhante ao apt/yum/pamac para Linux. Apesar da controvérsia sobre o modelo de gerenciamento de pacotes (já que softwares proprietários não são empacotados diretamente, então o chocolatey não pode ser chamado de "gerenciador de pacotes", o que para o usuário final dá na mesma), o Chocolatey quebra tanto o galho que eu acabo utilizando ele até para pesquisar softwares (antes mesmo de pesquisar no Google, já que os resultados são baseados na disponibilidade e rating dos softwares upados para a ferramenta e não de quem paga mais ads pro Google...). E se não bastasse, o chocolatey é a ferramenta mais utilizada em conjunto com configuration managers como Ansible/Puppet/Chef, então não precisa ter medo...

  • CLCL: Sendo o software mais simples e mais utilizado dessa lista, o CLCL permite fazer um cache dos últimos itens no seu clipboard e evitar que você perca aquele texto/imagem/arquivo que você copiou a pouco por N motivos. Em algum momento a Microsoft introduziu essa mesma funcionalidade no Windows, que pod ser acessada através do shortcut "win + v", então utilizar ou não um software que não updates a um bom tempo mas é tão simples que você realmente não precisa se preocupar sobre updates, fica a seu critério. o CLCL também é útil para colar texto sem formatação em editores WYSIWYG.

  • F.lux: Controla a luminosidade do seu monitor, regulando a temperatura das cores de acordo com o horário do dia. Parece uma besteira a princípio, mas desde que eu comecei a usar o F.lux, meus problemas com vista cansada ou falta de sono a noite diminuíram bastante. Se quiser entender mais sobre o assunto, pesquise sobre "led and circadian rhythm".

  • Enpass: Se você ainda não usa um gerenciador de senhas, já passou da hora de começar. Das várias opções do mercado, o Enpass na minha opinião é o melhor custo/beneficio, com uma interface muito parecia com o 1Password (que seria a minha primeira opção, não fosse a necessidade de assinatura para utilizar), ele armazena suas senhas no seu serviço de armazenamento de arquivos favorito (OneDrive, Google Drive, etc). Multi-plataforma, funciona muito bem em Windows e Android, estou tendo alguns problemas com a versão de Linux que não está carregando corretamente códigos TOTP (time-based one-time password), mas parece que já tem um fix disponível.

  • Vscode: Minha IDE padrão, utilizada tanto para programar quanto para edição de texto. Possui um vasto eco-sistema de plugins, alguns deles listados abaixo.

Ferramentas úteis por categoria

Não segui nenhuma regra aqui e as ferramentas listadas podem ser softwares que precisam ser instalados, plugins de outros softwares ou softwares apenas utilizadas via Web. Antes de começar a lista, apesar dos softwares citados aqui não serem focados em um tipo de uso específico, é bom citar que existe uma lista curada sobre ferramentas para DevOps em http://www.devopsbookmarks.com/.

Organização de tarefas / Documentação

  • Trello: Talvez o principal software para gerenciamento de tarefas/projetos atualmente. Melhor opção para sair usando sem precisar fazer grandes mudanças.

  • Notion: O Notion é uma ferramenta extremamente completa e flexível, não só pode ser utilizada para gerenciamento de tarefas e projetos como o Trello, mas também como uma ferramenta para documentação, criação de tabelas dinâmicas e muito mais. Em contra partida, o Notion tem tantas funcionalidades que é fácil perder o foco do que voê realmente precisa enquanto utilizando ele. Uma ótima opção ao OneNote para usuários de Linux.

  • Leankit / Planview: O Leankit veio parar nessa lista por um capricho meu, não é a melhor ferramenta no que se propõe e continua gratuíta somente para usuários antigos que possuem uma conta antiga, da época que era grátis. Contudo, após varias pesquisas, ainda não encontrei um substituto ao Leankit se tratando do layout de organização de tarefas, que se aproxima bastante de um quadro de kanban físico. É possível fazer algumas integrações para automatizar a criação de cards com softwares terceiros.

  • Todoist: Um Trello basado em listas. Tão poderoso como o Trello, pode se encaixar melhor ao seu tipo de organização, ou não.

Static site generators

A lista de softwares utilizados para geração de sites estáticos é grande e só vem aumentando com o passar do tempo. Como este artigo não deve receber atualizações tão cedo, os sites de referencia para pesquisar geradores de conteúdo estático são estes:

https://www.staticgen.com/ https://staticsitegenerators.net/ https://daveaglick.com/posts/a-survey-of-dotnet-static-site-generators

  • Wyam: O gerador utilizado neste blog. Leve, de fácil instalação e customização, utiliza .NET Core e pode ser instalado via choco ou pela cli do dotnet.

  • Jekyll: A ferramenta que popularizou novamente o uso de sites estáticos.

  • Statiq: Desenvolvido pelo mesmo criador do Wyam, o projeto está em desenvolvimento e já pode ser testado.

Vscode Extension:

  • Brazilian Portuguese - Code Spell Checker: Verificador de gramática para o VsCode. Permite escrever os posts do blog diretamente do editor, sem ter que passar num spell checker do Word por exemplo, além de ajudar com documentação de projetos.

  • REST Client: Permite executar chamadas REST diretamente do VsCode. Depois de utiliza-lo, vai ser difícil voltar para o Postman...

Chrome Extensions

  • Grammarly O Grammarly é um verificador de gramática para browsers, checa em tempo real quando você escreve numa caixa de texto, como por exemplo no Gmail. Mais do que apontar erros de gramatíca, essa ferramenta analisa a estrutura do seu texto e indica correções que ajudam a deixa-lo mais conciso.

  • The Great Suspender Todos sabemos se tem algo que o navegador Chrome gosta é da sua RAM. Essa extensão "hiberna" as guias abertas após determinado tempo sem atividade, economizando um pouco de memória durante o uso.

  • uBlock Origin

Debugging

  • Sysinternals e PsTools: Conjunto de ferramentas para analise e debugging do Windows.

  • Process Hacker: Semelhante ao Process Explorer do SysInternals, ajuda a identificar consumo de recursos e impersonar processos no Windows.

  • JustDecompile: Um Descompiler grátis, para .NET e afins.

  • Fiddler: Ferramenta para debugging de chamadas HTTP.

Network

  • Tcpdump for Windows: A versão para Windows do famoso TCPDump. Utiliza mesma syntax de comandos.

  • Wireshark: Ferramenta para debugging de rede, possui spec semelhante ao Tcpdump, mas inteiramente gráfica.
