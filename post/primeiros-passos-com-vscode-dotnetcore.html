Primeiros passos com Visual Studio Code e .Net Core

Published on Friday, May 8, 2020
dotnetcore vscode

Já a algum tempo eu queria estudar .Net Core e venho postergando. Confesso que a primeira experiência (a cerca de dois anos) não foi lá das melhores.

choco install vscode -y
choco install dotnetcore -y

Antes de começar o desenvolvimento, dei uma boa lida sobre os tipos de projeto possíveis de criar utilizando .Net Core e mesmo assim

WebApi (usa controlers de mvc): dotnet new webapi -o project Razor: dotnet new webapp -o projectName

new page inside a Razor project: dotnet new page -n Index -o Pages

Recuperar dados de arquivos de configuração somente via injection ou abrindo o appsettings.json explicitamente via IO https://www.fabiosilvalima.net/6-formas-appsettings-json-aspnet-core/

retrieving appsettings from class library

var configurationBuilder = new ConfigurationBuilder();
var path = Path.Combine(Directory.GetCurrentDirectory(), "appsettings.json");
configurationBuilder.AddJsonFile(path, false);
var root = configurationBuilder.Build();
Endpoint = root.GetSection("NasaData:Endpoint").Value;
